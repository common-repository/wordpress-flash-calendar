== Description ==

Add flash calendar in your wordpress.

== Installation ==

Upload `glll_wp_flash_calendar` to the `/wp-content/plugins/` directory. 
Change file name "date-sample.xml" to "date.xml".  
Edit "date.xml" for your thing. 
Place <?php glll_wp_flash_calendar(988); ?> or <?php glll_wp_flash_calendar(210); ?> in your templates~
